# 利用深度學習於屠宰場豬隻數量清點



### 簡介：
訓練YOLOv7模型偵測豬隻，並結合ByteTrack演算法追蹤偵測目標移動軌跡。最後將此系統部署至現場，藉由現場燈板即時顯示模型計算豬隻的總數量。

<br>

### 工具使用介紹：

:point_right:深度學習 : YOLOv7、ByteTrack
